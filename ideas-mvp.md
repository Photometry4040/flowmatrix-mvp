**Design Movement:** Neo-Brutalism meets Data Visualization Dashboard

**Core Principles:**
1. **Bold Geometric Structure** - 강렬한 테두리와 명확한 구획으로 정보 밀도 높은 인터페이스 구성

3. **Functional Maximalism** - 모든 UI 요소가 명확한 기능적 목적을 가지며 시각적으로 구분됨
4. **Grid-Based Canvas** - 매트릭스 구조를 시각적으로 강조하는 그리드 시스템

**Color Philosophy:**
- Base: 다크 배경 (#0a0a0a ~ #1a1a1a)으로 장시간 작업 시 눈의 피로 감소
- Primary: 사이버 블루 (#00d4ff)로 노드와 연결선 강조
- Accent: 네온 핑크 (#ff006e)로 병목 구간 경고
- Success: 라임 그린 (#39ff14)으로 AI 대체 가능 노드 표시
- 감정적 의도: 기술적, 미래지향적, 데이터 중심

**Layout Paradigm:**
- Full-Screen Canvas: 전체 화면을 캔버스로 활용, 최대한의 작업 공간 확보
- Floating Toolbars: 반투명 툴바가 캔버스 위에 떠있는 형태
- Sidebar Matrix Navigator: 좌측에 축소된 매트릭스 오버뷰로 빠른 네비게이션

**Signature Elements:**
1. **Pulsing Bottleneck Indicators** - 병목 노드가 붉은색으로 맥박치듯 애니메이션
2. **Neon Grid Lines** - 매트릭스 셀 구분선이 은은하게 발광
3. **Floating Node Cards** - 노드가 3D 효과로 떠있는 듯한 그림자

**Interaction Philosophy:**
- Direct Manipulation: 드래그 앤 드롭으로 직관적인 노드 배치
- Contextual Menus: 우클릭으로 컨텍스트 메뉴 즉시 표시
- Zoom & Pan: 무한 캔버스 탐색을 위한 부드러운 줌/팬

**Animation:**
- Smooth Transitions: 모든 상태 변화는 300ms ease-out
- Pulse Effect: 병목 노드는 1.5초 주기로 scale(1.05) 펄스
- Connection Flow: 연결선을 따라 흐르는 점선 애니메이션으로 데이터 흐름 표현

**Typography System:**
- Display: **Space Grotesk** (700) - 툴바와 패널 제목용
- Body: **Inter** (400, 500) - 노드 라벨과 설명
- Mono: **JetBrains Mono** (400) - 데이터 값과 통계
- Hierarchy: 큰 대비로 명확한 정보 계층 구축

## 선택된 디자인: Neo-Brutalism meets Data Visualization Dashboard

첫 번째 접근법을 선택합니다. 이유는 다음과 같습니다:

1. **시각적 임팩트**: 워크플로우 매핑 도구는 복잡한 데이터를 다루므로, 강렬한 시각적 구분이 사용자의 인지 부하를 줄여줍니다.

2. **병목 강조**: 네온 컬러와 펄스 애니메이션은 병목 구간을 즉각적으로 인지하게 만들어, PRD의 핵심 가치인 'Visibility'를 극대화합니다.

3. **장시간 사용성**: 다크 테마는 전사적 프로세스 매핑이라는 장시간 작업에 적합하며, 눈의 피로를 줄여줍니다.

4. **차별화**: 일반적인 화이트 캔버스 도구들과 차별화되어, 혁신적이고 기술 중심적인 이미지를 전달합니다.

5. **데이터 중심**: 높은 정보 밀도를 효과적으로 표현할 수 있는 디자인 언어로, AI 적합성 스코어, 리드타임 등 다양한 메트릭을 자연스럽게 통합할 수 있습니다.
