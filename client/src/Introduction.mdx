# FlowMatrix Storybook

FlowMatrix의 컴포넌트 카탈로그에 오신 것을 환영합니다!

이 Storybook은 FlowMatrix 프로젝트의 모든 UI 컴포넌트를 독립적으로 개발하고 테스트할 수 있는 환경을 제공합니다.

## 📚 컴포넌트 카테고리

### Components
FlowMatrix의 핵심 컴포넌트들입니다:

- **WorkflowNode**: 워크플로우 노드 (4가지 타입)
- **DraggableNodeType**: 드래그 가능한 노드 타입 카드
- **TagAutocomplete**: 온톨로지 태그 자동완성 입력
- **NodeDetailPanel**: 노드 상세 정보 패널 (추후 추가 예정)
- **MatrixView**: 매트릭스 뷰 레이아웃 (추후 추가 예정)

## 🎨 디자인 시스템

FlowMatrix는 **Neo-Brutalism meets Data Visualization Dashboard** 디자인 철학을 따릅니다:

### 색상 팔레트
- **Primary**: `oklch(0.65 0.25 230)` - 사이버 블루
- **Accent**: `oklch(0.60 0.30 350)` - 네온 핑크
- **Success**: `oklch(0.75 0.25 130)` - 라임 그린
- **Background**: `oklch(0.08 0.01 260)` - 다크 배경

### 타이포그래피
- **Display**: Space Grotesk (700)
- **Body**: Inter (400, 500)
- **Mono**: JetBrains Mono (400)

### 애니메이션
- **Smooth Transitions**: 300ms ease-out
- **Pulse Effect**: 1.5초 주기 scale(1.05)
- **Connection Flow**: 점선 애니메이션

## 🚀 사용 방법

1. 좌측 사이드바에서 원하는 컴포넌트를 선택합니다
2. 각 컴포넌트의 다양한 상태(Story)를 확인합니다
3. **Controls** 탭에서 props를 실시간으로 수정하여 테스트합니다
4. **Docs** 탭에서 컴포넌트 문서를 확인합니다
5. **Accessibility** 탭에서 접근성 검사 결과를 확인합니다
